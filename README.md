# Guessing-Game

Category-based Guessing Game inspired by jklm.fun, built with React and Supabase


## Getting Started
```bash
# Clone the repository
git clone https://github.com/yourusername/project-name

# Install dependencies
npm install

# Run the development server
npm run dev

```
